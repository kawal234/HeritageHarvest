# Heritage & Harvest — Premium Corporate Gifting

A modern, high-end landing page for Heritage & Harvest, specializing in premium corporate gifting. This project features dynamic 3D visualizations using Three.js and a sleek UI using custom CSS.

## File Structure

- `index.html`: The main entry point containing the semantic HTML structure.
- `src/style.css`: Contains all custom styling with CSS variables, animations, and responsive breakpoints.
- `src/main.js`: Main JavaScript logic handling Three.js 3D renderings (hero background and unboxing animation), scroll reveals, and form interactions.

## Setup & Run Locally

This project is built using Vite for a lightning-fast development experience.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Technologies Used

- **HTML5 & CSS3**: For structure, styling, CSS variables, and native animations.
- **JavaScript (ES6+)**: For interactivity and DOM manipulation.
- **Three.js**: To render the 3D particles and 3D gift box in the hero section, as well as the 3D unboxing preview visualization.
- **Vite**: Used as the frontend build tool for quick module replacement and optimized production builds.

## Deployment

To deploy this project (e.g., to GitHub Pages, Vercel, or Netlify), simply run `npm run build` and publish the generated `dist` folder.
